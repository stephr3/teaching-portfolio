Web Development Projects

Portland ESL Network

The Portland ESL Network is an organization dedicated to sharing information about ESL classes and resources in the Portland Metro area. The site helps students, teachers, and volunteers find classes in their neighborhoods.

My role:

I initially designed and developed the website using Ruby on Rails and deploying on Heroku. Later, in order to make the site easier for volunteers to update, I led a volunteer team of 3 to transition the site to WordPress, building a customized theme and creating customized map plugins.

Technologies used:

WordPress, PHP, JavaScript, Google Maps API

MyMizu

With over 200,000 refill spots worldwide, including spots added by users and partner shops and cafes that let the public refill for free - the MyMizu web app makes sure you never have to buy bottled water again.

My role:

I participated in a hackathon and pair programmed to add information overlays to the map portion of the web app.

Technologies used:

NodeJS, React, Google Maps API

SpringVoice

Spring is the first incorporated sexual assault survivors' organization in Japan. The organization is engaged in advocacy activities including optimizing the use of sex crime laws, as well as amendments to the law to make it as fair as possible to victims. The OneVoice app was designed to be a visual petition to encourage lawmakers and stakeholders to improve laws for sexual assault survivors.

My role:

I volunteered with the organization, "Siventh". I developed the app design with Figma, prepared image assets using Adobe Photoshop, and worked on the front end with React Native.

Technologies used:

Figma, React Native, Adobe Photoshop
